# Markdown Peek
Markdown Peek(`mdpeek`) is a markdown previewer and terminal renderer tool.

Supported Github Flavored Markdown(GFM), GitLab Flavored Markdown.

# Features
- Fast
- Live Reaload
- Easy Configure
- Supported GFM, GLFM and Terminal Preview
- Supperted Mermaid, PlantUML
- Supported KaTex
